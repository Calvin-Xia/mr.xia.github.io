## 改造 statement.html 为博客文章总展示页

### 主要修改内容：

1. **移除原有功能**

   * 删除 Markdown 渲染相关的 JavaScript 代码（MarkdownRenderer 对象）

   * 删除展开/收起按钮及相关功能

   * 移除 marked、highlight.js、KaTeX 等库的引用（保留页面基础样式）

2. **添加博客文章抓取功能**

   * 创建 JavaScript 代码动态获取 blog 文件夹下的所有 HTML 文件

   * 对每个 HTML 文件进行 fetch 请求

   * 解析 HTML 内容，提取：

     * `<div class="markdown-content">` 内的第一个 `<h1>` 标题

     * `<div class="markdown-content">` 内的第一个 `<p>` 正文内容

   * 将提取的内容以卡片形式展示在页面上

3. **更新页面结构**

   * 修改页面标题为"博客文章"或"文章列表"

   * 创建文章卡片容器，展示每篇文章的标题和摘要

   * 每个卡片包含链接，点击可跳转到对应文章页面

   * 添加加载状态提示和错误处理

4. **样式优化**

   * 为文章卡片添加统一的样式（阴影、圆角、悬停效果）

   * 确保响应式布局，适配不同屏幕尺寸

   * 保持与现有网站风格一致

### 实现方式：

* 使用 JavaScript 的 `fetch()` API 请求 blog 文件夹下的 HTML 文件

* 使用 DOMParser 解析 HTML 字符串

* 使用 `querySelector` 提取所需的 h1 和 p 标签内容

* 动态生成文章卡片并插入到页面中
