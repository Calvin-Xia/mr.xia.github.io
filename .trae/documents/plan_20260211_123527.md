# Markdown转HTML工具风格化计划

## 设计方向：日落橙粉玻璃态风格

将工具页面的视觉风格与主项目的日落橙粉玻璃态风格保持一致，同时维持单文件独立性。

***

## 修改范围（仅 markdown-to-html-tool.html）

### 1. CSS 变量更新（与主项目一致）

#### 主色调（日落暖色系列）

* `--primary-color`: #6366f1 → **#f97316** (珊瑚橙)

* `--primary-dark`: #4f46e5 → **#ea580c** (深珊瑚橙)

* `--primary-light`: #818cf8 → **#fb923c** (浅珊瑚橙)

* `--secondary-color`: #ec4899 → **#fb7185** (玫瑰粉)

#### 背景色（温暖柔和系列）

* `--background-color`: #ffffff → **#fef7ed** (暖米白)

* `--bg-primary`: #ffffff → **rgba(255, 255, 255, 0.75)** (半透明白)

* `--bg-secondary`: #f9fafb → **rgba(254, 247, 237, 0.85)** (暖浅灰)

#### 边框与交互色

* `--border-color`: #e5e7eb → **rgba(249, 115, 22, 0.2)** (暖橙半透明)

* `--hover-color`: #f3f4f6 → **rgba(249, 115, 22, 0.1)** (暖橙淡色)

#### 背景渐变（日落流动）

* 原渐变：#667eea → #764ba2 → #f093fb → #4facfe → #00f2fe

* 新渐变：**#fed7aa → #fecdd3 → #a5f3fc → #fdba74 → #fed7aa**

***

### 2. 样式增强（玻璃态效果）

#### 添加玻璃态效果

* `.editor-panel`: 添加 `backdrop-filter: blur(20px) saturate(180%)`

* `.preview-content`: 添加玻璃态背景

* `.help-section`: 添加玻璃态效果

#### 添加装饰性圆形

* `.bg-img::before` 和 `.bg-img::after`: 添加漂浮的渐变圆形

#### 优化阴影和边框

* 使用新的日落橙色系阴影

* 更新所有边框颜色为暖橙半透明

***

### 3. 导出HTML样式更新（generateFullHTML函数）

#### 导出文件的样式变量

* 更新所有 CSS 变量为日落橙粉配色

* 添加玻璃态效果到导出样式

* 更新背景渐变为日落风格

#### 导出文件的背景

* 添加日落渐变背景

* 添加玻璃态卡片效果
