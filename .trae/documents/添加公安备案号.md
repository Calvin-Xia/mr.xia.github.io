# 在ICP备案号下方添加公安备案号

## 任务概述

在index.html的页脚区域，ICP备案号链接下方添加公安备案号，包含公安徽标图片。

## 实施步骤

### 1. 查看当前页脚结构

当前ICP备案号位置：
```html
<div class="footer">
    <p class="footer-text">
        <a href="https://beian.miit.gov.cn/" target="_blank" rel="noopener">渝ICP备2026000319号</a>
    </p>
</div>
```

### 2. 添加公安备案号

在ICP备案号下方添加：
```html
<div class="footer">
    <p class="footer-text">
        <a href="https://beian.miit.gov.cn/" target="_blank" rel="noopener">渝ICP备2026000319号</a>
        <br>
        <a href="https://beian.mps.gov.cn/#/query/webSearch?code=50010102001439" rel="noreferrer" target="_blank">
            <img src="https://beian.mps.gov.cn/web/assets/logo01.6189a29f.png" alt="公安备案" style="width: 20px; height: 20px; vertical-align: middle; margin-left: 5px;">
            渝公网安备50010102001439号
        </a>
    </p>
</div>
```

### 3. 样式调整

添加CSS样式确保公安徽标正确显示：
- 图片大小：20px × 20px
- 垂直居中：vertical-align: middle
- 左边距：5px
- 与ICP备案号在同一行

## 预期效果

✅ ICP备案号下方显示公安备案号
✅ 公安徽标图片正确显示
✅ 链接可点击跳转到公安备案查询网站
✅ 页脚样式保持统一
