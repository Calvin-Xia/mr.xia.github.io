## 资源预加载实施方案

### 修改文件列表

**通用页面** (添加DNS预解析 + CSS预加载):
1. `index.html`
2. `about.html`
3. `Works.html`
4. `timetable.html`
5. `statement.html`

**特殊页面** (额外添加CDN资源预加载):
7. `markdown-to-html-tool.html` - 预加载 marked.js, highlight.js, KaTeX

### 添加内容

```html
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
    <!-- DNS预解析 -->
    <link rel="preconnect" href="https://cdnjs.cloudflare.com">
    <link rel="dns-prefetch" href="https://cdnjs.cloudflare.com">
    
    <!-- CSS预加载 -->
    <link rel="preload" href="css/style.css" as="style">
    
    <!-- 原有内容 -->
    ...
</head>
```
