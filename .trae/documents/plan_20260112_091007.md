# 实现无刷新多级导航系统

## 任务概述

使用HTML5 History API实现无缝导航体验，无需页面刷新即可在不同页面间切换。

## 实施步骤

### 1. 创建统一的导航模块

**创建文件**：js/navigation.js

**核心功能**：

```javascript
// 页面状态管理
const PageStateManager = {
    states: new Map(),
    saveState(pageId, state) {
        this.states.set(pageId, state);
    },
    getState(pageId) {
        return this.states.get(pageId) || {};
    }
};

// 导航控制器
const NavigationController = {
    goHome() {
        const homeState = PageStateManager.getState('index');
        history.pushState({ from: this.getCurrentPage(), restored: true }, '', 'index.html');
        this.restoreHomeState(homeState);
    },
    
    handlePopState(event) {
        const state = event.state;
        if (state && state.restored) {
            this.restoreHomeState(state.data);
        }
    },
    
    saveCurrentState() {
        const scrollY = window.scrollY;
        const pageId = this.getCurrentPage();
        PageStateManager.saveState(pageId, { scrollY });
    },
    
    getCurrentPage() {
        return window.location.pathname.split('/').pop() || 'index.html';
    }
};
```
