## 实现文件名时间戳提取和展示功能

### 主要修改内容：

1. **添加时间戳提取函数**
   * 创建 `extractTimestampFromFileName(fileName)` 函数
   * 使用正则表达式 `^(\d{8})` 匹配文件名开头的8位数字
   * 验证提取的数字是否为有效日期格式（YYYYMMDD）
   * 容错处理：如果匹配失败或日期无效，返回 null

2. **添加时间戳格式化函数**
   * 创建 `formatTimestamp(timestamp)` 函数
   * 将8位数字字符串解析为 Date 对象
   * 格式化为标准日期格式 `YYYY-MM-DD`
   * 容错处理：如果日期解析失败，返回 null

3. **修改 loadArticle 方法**
   * 在提取文件名后，调用时间戳提取函数
   * 调用格式化函数获取格式化后的日期字符串
   * 将日期信息添加到返回的 article 对象中

4. **修改 createArticleCard 方法**
   * 在卡片底部添加日期展示元素
   * 使用 `<div class="blog-card-date">` 包裹日期
   * 确保日期在卡片内部、链接下方、左对齐显示

5. **添加日期样式**
   * 为 `.blog-card-date` 添加样式
   * 设置字体大小、颜色、间距等
   * 使用次要文字颜色，与标题和摘要区分
   * 响应式适配

### 实现细节：

* 正则表达式：`/^(\d{8})/` 提取文件名开头的8位数字
* 日期验证：检查月份在 1-12 范围内，日期在合理范围内
* 日期格式化：使用 JavaScript Date 对象和模板字符串
* 错误处理：使用 try-catch 和条件判断确保容错性
* 样式：使用 CSS 变量保持与现有设计风格一致
