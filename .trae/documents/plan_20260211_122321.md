# CSS 配色优化计划

## 设计方向：日落橙粉渐变风格

采用温暖、柔和的日落色调，营造温馨优雅的视觉体验，避免常见的蓝紫渐变。

---

## 修改范围（仅 css/style.css）

### 配色方案调整

#### 主色调（温暖日落系列）
- `--primary-color`: 珊瑚橙 → **#f97316**
- `--primary-dark`: 深珊瑚橙 → **#ea580c**
- `--primary-light`: 浅珊瑚橙 → **#fb923c**
- `--secondary-color`: 玫瑰粉 → **#fb7185**
- `--accent-color`: 薄荷绿 → **#34d399**
- `--error-color`: 柔和红 → **#f87171**

#### 背景色（温暖柔和系列）
- `--background-color`: 暖米白 → **#fef7ed**
- `--bg-primary`: 半透明白 → **rgba(255, 255, 255, 0.75)**
- `--bg-secondary`: 暖浅灰 → **rgba(254, 247, 237, 0.85)**

#### 边框色（温暖系列）
- `--border-color`: 暖橙半透明 → **rgba(249, 115, 22, 0.2)**

#### 交互色（温暖系列）
- `--hover-color`: 暖橙淡色 → **rgba(249, 115, 22, 0.1)**

#### 文字色（保持深灰确保可读性）
- `--text-primary`: 深灰 → **#374151**
- `--text-secondary`: 中灰 → **#6b7280**
- `--text-light`: 浅灰 → **#9ca3af**

#### 背景渐变（日落流动）
- 淡桃色 (#fed7aa) → 淡玫瑰色 (#fecdd3) → 淡青色 (#a5f3fc) → 淡橙色 (#fdba74)

#### 装饰性圆形渐变
- 圆形1：珊瑚橙 (#f97316) → 玫瑰粉 (#fb7185)
- 圆形2：薄荷绿 (#34d399) → 琥珀色 (#fbbf24)
