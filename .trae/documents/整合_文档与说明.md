- **marked.js v11.1.1**: Markdown解析器
  - 支持GFM（GitHub Flavored Markdown）
  - breaks配置选项
  - 完整的Markdown语法支持

- **highlight.js v11.9.0**: 代码语法高亮
  - 支持多种编程语言
  - github-dark主题
  - 自动语言检测

- **KaTeX v0.16.9**: 数学公式渲染
  - 支持$和$$分隔符
  - 高性能渲染
  - 完整的数学符号支持

#### 配置说明
```javascript
marked.setOptions({
    breaks: false,  // 保持Markdown标准行为
    gfm: true      // 启用GitHub Flavored Markdown
});
```

### 博客系统文档

#### JSON配置文件

**blog-files.json格式**
```json
[
    "blog/20251231-2025年度总结.html",
    "blog/20250101-新年计划.html"
]
```

**blog-metadata.json格式**
```json
[
    {
        "id": "001",
        "title": "文章标题",
        "excerpt": "文章摘要",
        "category": "技术",
        "tags": ["标签1", "标签2"],
        "date": "2025-01-01",
        "filePath": "blog/20250101-文章标题.html"
    }
]
```

#### 使用说明
1. 创建新的HTML文章文件
2. 在blog-files.json中添加文件路径
3. 在blog-metadata.json中添加文章元数据
4. 系统自动加载并显示文章

### 导航系统文档

#### HTML5 History API
- pushState(): 添加历史记录
- replaceState(): 替换当前历史记录
- popstate事件：监听历史记录变化

#### 状态管理
- 使用Map存储页面状态
- sessionStorage临时存储
- 自动保存和恢复滚动位置
- 表单输入状态保留

#### 使用示例
```javascript
// 导航到新页面
history.pushState({ page: 'works' }, '', '#works');

// 处理后退/前进
window.addEventListener('popstate', (event) => {
    const state = event.state;
    if (state && state.page) {
        navigateTo(state.page);
    }
});
```
