📖 使用说明

功能特性

  • 支持完整的Markdown语法（GFM）
  • 代码语法高亮（highlight.js + github-dark主题）
  • 数学公式渲染（KaTeX，支持 $ 和 $$ 分隔符）
  • 图片自动添加备注文本（从alt属性提取）
  • 折叠/展开预览功能
  • 图片智能预加载
  • 导出为独立HTML文件
  • 响应式设计，支持移动端

使用方法

1. 在左侧输入框中输入或粘贴Markdown内容

2. 点击"渲染"按钮或使用快捷键（Ctrl+Enter）进行渲染

3. 右侧预览区域将显示渲染后的HTML效果

4. 点击"导出HTML"可将结果保存为独立HTML文件

5. 点击"复制HTML"可将HTML代码复制到剪贴板

文本换行说明

本工具使用标准Markdown语法，文本换行需要使用两个空格+换行：

这是第一行  
这是第二行

注意：单个换行符不会产生换行效果，需要使用两个空格+换行来实现。

快捷键
