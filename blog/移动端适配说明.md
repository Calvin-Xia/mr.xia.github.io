# HTML文件移动端适配说明

## 问题
dooc社区导出的HTML文件未动态适配手机端浏览器。

## 原因
缺少viewport meta标签，移动端浏览器无法正确识别设备宽度。

## 解决方案

在每个HTML文件的 `<head>` 标签内添加viewport声明。

### 修改位置
找到 `<meta charset="utf-8">` 这一行，在其后添加一行。

### 修改前
```html
<head>
  <meta charset="utf-8">
  <title>页面标题</title>
```

### 修改后
```html
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>页面标题</title>
```

## 说明
| 属性 | 作用 |
|------|------|
| `width=device-width` | 页面宽度等于设备屏幕宽度 |
| `initial-scale=1.0` | 初始缩放比例为1（不缩放） |
