# Phase 2.5 Code Review Checklist

## 图片悬浮放大
- [ ] 点击文章图片后打开悬浮层
- [ ] 悬浮层显示图片和灰色说明文字
- [ ] Escape、关闭按钮、点击遮罩均可关闭
- [ ] 关闭后焦点回到原图片
- [ ] 放大、缩小、重置按钮可用
- [ ] 缩放范围限制在 1x–4x
- [ ] 图片在桌面和移动端均不溢出视口
- [ ] `prefers-reduced-motion: reduce` 下关闭非必要动画

## 文章平滑切换
- [ ] `/articles/` 到文章详情页具备平滑切换
- [ ] 文章详情页之间具备平滑切换
- [ ] 返回文章列表不破坏浏览器历史记录
- [ ] 不支持 View Transitions 的浏览器回退为普通链接跳转
- [ ] 页面切换后图片 caption、灯箱、标题锚点、侧栏目录重新初始化
- [ ] 全局事件监听不会重复注册
- [ ] 外链和下载链接不被文章切换逻辑接管

## 标题层级与锚点
- [ ] `.markdown-content h1-h4` 层级间距、字号、字重清晰
- [ ] 标题字距为非负值，且不随 viewport width 缩放
- [ ] 标题 hover/focus 时显示 `#` 锚点
- [ ] 锚点可点击并更新 URL hash
- [ ] 中文标题、英文标题、带标点标题均生成稳定 id
- [ ] 重复标题不会产生重复 id
- [ ] 键盘用户可以 focus 到标题锚点

## 侧栏目录与进度
- [ ] 长文章桌面端显示侧栏目录
- [ ] 短文章（少于 3 个标题）隐藏侧栏目录
- [ ] 目录按 `h2/h3/h4` 层级缩进
- [ ] 点击目录项平滑滚动到对应标题
- [ ] 滚动时当前章节高亮
- [ ] 阅读进度准确显示 0–100%
- [ ] 移动端目录折叠，不遮挡正文
- [ ] 侧栏不与正文、图片、页脚重叠

## 自动化与构建
- [ ] `npm test` 覆盖灯箱、标题 id、目录数据、阅读进度核心逻辑
- [ ] `npm run test:coverage` 通过
- [ ] `npm run build` 通过且无新增警告
- [ ] 浏览器验证桌面宽度、平板宽度和手机宽度
- [ ] Phase 2.5 文档状态已更新
